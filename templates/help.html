How to write the resource

Each resource help page is a HTML file in template/resource folder. File name should be in the help_page_name.html form. Our Jupyter extension call for help page depend on current Jupyter notebook name with file extension, so for First.ipynb notebook help file should be named as first.ipynb.html.

Cell code and other parameters from extension is sended to server as ?cell_code=start_section form, and pulled to html template for rendering with Jinja template engine as dictionary. Any correct Jinga tags is allowed.
